# Preliminaries

### Step One (Variable Extract)
1. Please create a structure called mimic3_sepsis_new in MySQL.
2. You will need to run Variable_Extract.sql to extract the data based on the clinical indicator's corresponding itemid in MIMIC III.

### Step Two (Variable Clean)
1. Please create a structure called mimic3_sepsis_cleaned in MySQL.
2. You will need to run Variable_Clean.sql clean the wrong data from the clinical data screened in the previous step and to perform unit unification.
